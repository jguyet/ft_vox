# ft_vox
ft_vox est un projet d'introduction au moteur de voxel, inspiré de minecraft, dans lequel vous devez créer un monde généré aléatoirement.

##### devices :
Windows 32 (build OK)  
Linux (build OK)  
MacOS (build OK)  

##### require :
java 1.8
maven 3.6.3

##### use :
Compilation command :  
`mvn clean package`  
`./run.sh`
